# @attraction/icons &middot; ![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)

어트랙션에서 사용되는 아이콘들을 관리하고 리액트 컴포넌트를 생성하는 라이브러리에요.

## Features

`svgo`와 `svgr`라이브러리를 기반으로 관리중인 아이콘에 대하여 리액트 컴포넌트를 생성해요.
